# Security policy

## Supported versions

Security fixes are made for the latest released version of ai-cli-control.

## Reporting a vulnerability

Do not include credentials, access tokens, private logs, or account details in a public issue. Report the issue privately to the repository owner with a minimal reproduction, impact, and affected version. Allow time for assessment before public disclosure.

This project launches third-party CLIs. Provider account, authentication, model access, and service vulnerabilities must also be reported through the affected provider's security process.
